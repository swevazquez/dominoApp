# Chiva Support

Chiva is a minimal dominoes score tracker for recording hands, totals, and completed game summaries.

## Help

If a score looks incorrect, check the selected game preset and review the hand breakdown in the game summary. You can undo the most recent active-game hand or start a new game from the main screen.
